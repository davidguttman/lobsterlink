LobsterLink Bridge

Automation Control Panel

Deterministic extension tab for driving host lifecycle, viewer launch, tab switching, viewport control, and diagnostics without using the popup.

Agent Steps

Use This Order
  1. Select the website tab in Target Tab. Waiting
  2. Click Start Host. The hosted tab auto-focuses. Waiting
  3. Read the host ID and viewer URL from this bridge page: Current Peer ID and the Viewer URL field. If the bridge is no longer frontmost, reopen bridge.html — current host state persists and these fields will still be populated. Waiting
  4. Keep the hosted tab active/frontmost for the duration of the session. Waiting
Required: The hosted tab must stay the active tab in its window. Start Host focuses it automatically; backgrounded hosted tabs stall CDP screencast and produce a black viewer.
Source of truth: the bridge page holds the authoritative host ID and viewer URL. Reopen bridge.html at any time to recover them — do not rely on any host-tab overlay to read the ID.
Black viewer recovery: bring the hosted tab to the front (use Show Hosted Tab) before investigating anything else.
